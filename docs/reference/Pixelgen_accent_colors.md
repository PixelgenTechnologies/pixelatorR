# Pixelgen accent colors

With a soft base we allow accent colors to pop and reveal important data
and insights without complicating the picture. Accent colors are meant
to be used for data visualisation or as small touches in a composition.
The accent colors can also be used sparingly to emphasize clickable
content and to create visual hierarchy.

## Usage

``` r
Pixelgen_accent_colors
```

## Format

A tibble with 12 rows and 11 columns
