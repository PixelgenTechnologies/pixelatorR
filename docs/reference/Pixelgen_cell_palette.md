# Pixelgen cell type colors

This vector contains Pixelgen branded colors that are assigned to
specific cell types. These colors can be used in visualizations to
maintain consistency with Pixelgen branding and to help differentiate
between cell types in plots such as UMAPs, bar plots, or heatmaps. The
colors are chosen to provide good contrast between different cell types
while adhering to the Pixelgen color palette.

## Usage

``` r
Pixelgen_cell_palette
```

## Format

A named character vector where names are cell types and values are color
hex codes.
