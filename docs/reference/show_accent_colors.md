# Show accent colors

This function displays the Pixelgen accent colors in a grid format.

## Usage

``` r
show_accent_colors(label_discrete_order = FALSE)
```

## Arguments

- label_discrete_order:

  Logical, the color order used for
  [`color_discrete_pixelgen`](color-themes.md) are shown in the plot.

## Value

A ggplot object displaying the accent colors.
