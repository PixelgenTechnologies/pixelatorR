# Global Options in pixelatorR

Information about global options provided in pixelatorR.

## Details

Available options:

- `pixelatorR.verbose` Whether to print messages to the console.

## Verbosity

Verbosity can be turned off globally for all pixelatorR function by
setting `options(pixelatorR.verbose = FALSE)`.

## Examples

``` r
# Turn off verbosity
options(pixelatorR.verbose = FALSE)
```
