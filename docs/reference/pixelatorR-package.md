# pixelatorR: Data Structures, Data Processing Tools and Visualization Tools for MPX Single Cell Data

Defines S4 classes for single-cell MPX data, extending the Seurat
toolbox. In addition, pixelatorR provides functions to process and
visualize MPX data.
