# The pixelatorR style

Style code according to the pixelatorR style guide. This style guide is
based on the tidyverse style guide, but with some modifications.

## Usage

``` r
pixelatorR_style()
```

## Value

A list of transformers that can be used with e.g.
[`styler::style_text()`](https://styler.r-lib.org/reference/style_text.html)

## Details

- `unindent_fun_dec` is set to `NULL`

- `remove_line_breaks_in_fun_dec` is set to `NULL`
